computeFeatures_torch(I1, I2, I3, Ia, Ib):

Compute the features dependent on the right Cauchy-Green strain invariants. Note that the features only depend on I1 and I3 for plane strain assumption.

Input Arguments

  • I1 - 1st invariant

  • I2 - 2nd invariant

  • I3 - 3rd invariant

  • Ia - Invariant along anisotropy direction.

  • Ib - Invariant along second anisotropy direction.

Output Arguments

  • x - features
